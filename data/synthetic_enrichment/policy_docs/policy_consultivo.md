# Política Comercial — Mensagem Consultiva (sintético/fictício)

Este documento é fictício, criado para fins de demonstração do Datathon 7MLET.

## Elegibilidade

- Aplicável a qualquer cliente elegível à campanha, sem restrição de produto.
- Especialmente indicada para clientes com `default = unknown`, já que não é
  uma oferta de crédito ou investimento direto — reduz risco de suitability
  para o segmento com informação incompleta.

## Suitability

- Esta mensagem não deve ser contabilizada como "oferta comercial" para fins de
  limite de contatos por campanha (`campaign`), pois tem caráter educativo, não
  transacional.

## Observações de compliance

- A mensagem consultiva não pode conter nenhuma menção a produto específico,
  taxa ou condição comercial — é puramente orientativa sobre reserva de
  emergência.
